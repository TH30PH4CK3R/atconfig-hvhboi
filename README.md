### How to install?
open file manager, press <kbd>CTRL</kbd>+<kbd>H</kbd>

now copy & paste <kbd>hvh</kbd> <kbd>legit</kbd> <kbd>rage</kbd> folders to
```
~/.config/AimTux
```
